# UE5-AI

学习一下UE5.7的人工智能。   
目标暂定为制作一个武打片的场景。    
要求：  
1.多个AI，暂定7个近战，3个远程  
2.最多3个近战AI同时攻击     
3.远程AI能利用地形和掩体躲避    
4.当有近战AI死亡时，空闲的近战AI需要补位    
进阶场景：  
闹市街头（很多无攻击行为的NPC），玩家闲逛，突然触发什么，然后有攻击行为的NPC进行刺杀。  
无攻击行为的NPC需要panic，逃跑，躲避。
